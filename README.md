# vmware-vcenter-ansible

Ansible automation for VMware vCenter — inventory gathering.

## Contents

| File | Purpose |
|------|---------|
| `vcenter-gather-inventory.yml` | Gather vCenter VM and host inventory |

## Prerequisites

- Ansible 2.12+
- VMware collection: `ansible-galaxy collection install community.vmware`

## CI/CD

ansible-lint, secret scan, header compliance.
